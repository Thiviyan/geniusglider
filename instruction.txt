Geniusglider IT Services

### Executive Brief
Create a conversion-optimized, enterprise-grade landing page for Geniusglider, an IT services company targeting SMBs and mid-market clients globally. Build using modern web standards with advanced UX/UI, micro-interactions, progressive enhancement, and data-driven conversion optimization.

### Technical Architecture & Performance
- **Framework**: Vanilla HTML5/CSS3/ES6+ with progressive enhancement
- **Performance targets**: Lighthouse 95+ across all metrics, Core Web Vitals optimized
- **Loading strategy**: Critical CSS inlined, resources preloaded, lazy loading for non-critical assets
- **Bundle optimization**: Tree-shaking, code splitting, compression (Brotli/Gzip)
- **Caching**: Service worker implementation for offline functionality and asset caching
- **Image optimization**: WebP/AVIF with fallbacks, responsive images with `srcset`
- **Font loading**: `font-display: swap` with system font fallbacks during load

### Advanced Design System & Visual Hierarchy
**Color Palette (WCAG AAA compliant)**:
- Primary: `#0A1B35` (Navy) - 13.2:1 contrast ratio
- Secondary: `#1DB584` (Emerald) - accent and CTA
- Tertiary: `#F8FAFC` (Ghost) - backgrounds
- Semantic: Success `#059669`, Warning `#D97706`, Error `#DC2626`
- Text hierarchy: `#0F172A`, `#475569`, `#64748B`, `#94A3B8`

**Typography Scale (Modular)**:
- Base: 16px, Scale: 1.25 (Major Third)
- Headings: Custom geometric sans (Manrope/Inter alternative)
- Body: Optimized system stack with fallback cascade
- Code: JetBrains Mono for technical content

**Spacing System**: 4px base unit, exponential scale (4, 8, 12, 16, 24, 32, 48, 64, 96, 128px)

**Micro-interactions & Animation**:
- Scroll-triggered animations using Intersection Observer API
- Staggered card reveals with spring physics (cubic-bezier easing)
- Morphing navigation states with FLIP technique
- Loading states with skeleton screens
- Form validation with real-time feedback animations
- Hover states with magnetic button effects
- Parallax hero elements with performance optimization
